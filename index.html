Intro

สวัสดีครับ ผมชื่อนายนิรุทธ์ โพธิ เป็นคนที่มีความหลงใหลในเทคโนโลยีและการพัฒนาเว็บ ด้วยประสบการณ์ในการเขียนโปรแกรมและการทดสอบซอฟต์แวร์ ฉันชอบที่จะเรียนรู้สิ่งใหม่ ๆ และหาทางแก้ปัญหาที่ท้าทาย

นอกจากการเขียนโค้ดแล้ว ฉันยังมีความสนใจในการออกแบบและสร้างประสบการณ์ที่ดีที่สุดให้กับผู้ใช้ ฉันเชื่อว่าคุณภาพของซอฟต์แวร์มีความสำคัญอย่างยิ่งต่อความสำเร็จของโปรเจกต์ และฉันมุ่งมั่นที่จะทำงานร่วมกับทีมเพื่อให้บรรลุเป้าหมายนี้

Work Interest

การเขียนโค้ด (Coding)

ฉันมีความสนใจอย่างมากในงานเขียนโค้ดและการพัฒนาซอฟต์แวร์ ฉันชอบที่จะหาทางแก้ปัญหาที่ซับซ้อนและสร้างแอปพลิเคชันที่มีประสิทธิภาพและสามารถขยายได้ ประสบการณ์ของฉันรวมถึงการเขียนโปรแกรมในหลายภาษา เช่น JavaScript, Python และ Java โดยเฉพาะอย่างยิ่งฉันสนใจในงานพัฒนาเว็บไซต์ ซึ่งฉันสามารถรวมทักษะทั้งในด้าน Front-end และ Back-end เพื่อสร้างเว็บไซต์ที่มีความไดนามิกและใช้งานง่าย

การทดสอบ (Tester)

ในฐานะผู้ทดสอบ ฉันมีความสนใจในการรับรองคุณภาพของซอฟต์แวร์ โดยมุ่งเน้นที่การตรวจสอบและประเมินซอฟต์แวร์เพื่อให้แน่ใจว่ามีการทำงานอย่างถูกต้องและไม่มีข้อบกพร่อง ฉันสนใจที่จะเรียนรู้และใช้เครื่องมือการทดสอบที่ทันสมัย เช่น Selenium และ JUnit เพื่อลดความผิดพลาดและเพิ่มประสิทธิภาพในกระบวนการทดสอบ นอกจากนี้ ฉันยังให้ความสำคัญกับการทำงานร่วมกับทีมพัฒนา เพื่อให้สามารถสื่อสารปัญหาหรือข้อเสนอแนะได้อย่างมีประสิทธิภาพ ซึ่งช่วยให้ผลิตภัณฑ์ที่ส่งมอบมีคุณภาพสูงและตอบสนองความต้องการของผู้ใช้ได้อย่างแท้จริง

About

สวัสดีครับ ผมชื่อ นายนิรุทธ์ โพธิ ชื่อเล่น บอล อายุ 24 ปี เกิดวันที่ 20 เมษายน พ.ศ.2543  ที่อยู่ บ้านเลขที่ 190 หมู่ 7 ตำบลแวง อำเภอสว่างแดนดิน จังหวัดสกลนคร รหัสไปรษณีย์ 47240

การศึกษา

-ปี 2560 จบ ปวช. จาก วิทยาลัยเทคโนโลยีสว่างแดนดิน สาขาวิชา คอมพิวเตอร์ธุรกิจ
-ปี 2562 จบ ปวส. จาก วิทยาลัยการอาชีพสว่างแดนดิน สาขาวิชา คอมพิวเตอร์ธุรกิจ
-ปัจจุบัน กำลังศึกษาในระดับปริญญาตรี สาขาวิชา วิศวกรรมคอมพิวเตอร์และการสื่อสาร
